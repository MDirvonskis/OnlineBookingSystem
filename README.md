# OnlineBookingSystem

# Installation Guide:
Express: https://expressjs.com/
```
npm init -y
npm i express
```

Vue.js: https://vuejs.org/
```
npm install vue 
npm install vue@csp
```
mySQL2: https://www.npmjs.com/package/mysql2
```
npm i mysql2
```

# Start Server
```
node server.js
```
Notes: To start the server, you need to be in /app route.
