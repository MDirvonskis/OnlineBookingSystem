# OnlineBookingSystem

# Start server
```
node server.js
```

# Installation Guide:

Install Express.js :https://expressjs.com/
```
npm init
npm install express
```

Install Vue.js :https://vuejs.org/
```
npm install vue 
npm install vue@csp
```

Install mySQL :https://www.npmjs.com/package/mysql2
```
npm i mysql2
```
