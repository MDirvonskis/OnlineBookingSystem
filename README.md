# OnlineBookingSystem

# Installation Guide:
express:
npm init -y
npm i express
npm i --save-dev nodemon

react:
npm install -g create-react-app
npx create-react-app
npm install react-router
npm i -S react-scripts

mySQL Database:
npm i mysql2
